## Приложение А, приоритеты и порядок разбора операторов

|                  Операторы                   |  Разбираются  |
|----------------------------------------------|---------------|
| :---                                         |     :---:     |
| ()  []  ->  .                                | слева направо |
| !  ~  ++  --  +  -  *  &  (type)  sizeof     | справа налево |
| *  /  %                                      | слева направо |
| +  -                                         | слева направо |
| <<  >>                                       | слева направо |
| <  <=  >  >=                                 | слева направо |
| ==  !=                                       | слева направо |
| &                                            | слева направо |
| ^                                            | слева направо |
| \|                                           | слева направо |
| &&                                           | слева направо |
| \|\|                                         | слева направо |
| ?:                                           | справа налево |
| =  +=  -=  \*=  /=  %=  &=  ^=  \|=  <<=  >>=| справа налево |
| ,                                            | слева направо |
